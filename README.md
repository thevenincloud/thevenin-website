# Thevenin Landing

A modern Next.js landing page for Thevenin's cloud infrastructure platform.

## Features
- Striking hero section with Thevenin theorem animation
- Cookie consent banner (GDPR compliant)
- Legal pages: Privacy Policy & Terms of Service
- Responsive design with Tailwind CSS
- Minimalist infrastructure metaphors
- Pre-seed investor page with pitch deck link

## Getting Started
1. Install dependencies:
   ```bash
   pnpm install
   ```
2. Run the development server:
   ```bash
   pnpm dev
   ```
3. Visit [http://localhost:3000](http://localhost:3000) in your browser.

## Project Structure
- `app/` — Next.js App Router pages
- `components/` — UI and layout components
- `hooks/` — Custom React hooks
- `lib/` — Utility functions
- `public/` — Static assets
- `styles/` — Global and utility CSS
